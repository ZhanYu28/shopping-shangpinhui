# shopping-shangpinhui
这是一个使用vue3+vuex制作而成的pc端购物平台项目

## 开发环境
+ 系统：win7
+ ndoejs：12.14.0
+ vue-cli：5.0.0

## 使用到的插件第三方库等
+ axios
+ 加载进度条：nprogress
+ 组件库：element-plus
+ less
+ 模拟数据：mockjs
+ 随机id：nanoid
+ 轮播图：swiper
+ 表单验证：vee-validate
+ 图片懒加载：vue-lazyload

## Project setup
```
npm install
```
